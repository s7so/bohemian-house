## ✨ About The Project

> *"Bohemian House transforms spaces into sanctuaries — where earthy materials, organic forms, and timeless elegance converge."*

**Bohemian House** is a premium eco-friendly interior design studio based in **New Cairo, Egypt**, with branches in **Riyadh 🇸🇦** and **Luxembourg 🇱🇺**. Our digital portfolio showcases our philosophy of blending nature-inspired design with modern aesthetics.

## 🏗️ Project Structure

```
bohemian-house/
│
├── 📄 pages/
│   ├── 🏠 Home.jsx           — Landing page with full sections
│   ├── 🖼️  Portfolio.jsx      — Filterable project gallery
│   └── 🔐 Admin.jsx          — Password-protected dashboard
│
├── 🧩 components/
│   ├── HeroSection           — Parallax hero with image slideshow
│   ├── ServicesSection       — 3D tilt service cards
│   ├── FeaturedProjects      — Masonry project grid
│   ├── TestimonialsSection   — Animated testimonial carousel
│   ├── VideosSection         — YouTube video gallery
│   ├── StatsSection          — Animated number counters
│   ├── ContactSection        — Contact form + info
│   ├── Navbar                — Responsive navigation
│   ├── Footer                — Branded footer
│   └── WhatsAppButton        — Floating WhatsApp CTA
│
└── 🗃️ entities/
    ├── Project               — Portfolio projects schema
    ├── Service               — Services schema
    ├── Testimonial           — Client testimonials schema
    └── ContactMessage        — Inquiry messages schema
```

## 🌍 Pages & Features

### 🏠 Home Page
- ✅ Parallax hero with animated image slideshow + mouse tracking
- ✅ 3D tilt interactive service cards
- ✅ Featured projects masonry grid with scroll parallax
- ✅ YouTube video gallery
- ✅ Animated statistics counter
- ✅ Testimonials carousel with swipe animation
- ✅ Contact form with real-time DB storage
- ✅ Floating WhatsApp CTA button

### 🖼️ Portfolio Page
- ✅ Full project gallery with category filtering
- ✅ Lightbox modal with image gallery
- ✅ Masonry-style animated layout

### 🔐 Admin Dashboard
- ✅ Password-protected access
- ✅ Full CRUD for projects (add, edit, delete, feature toggle)
- ✅ Image upload support
- ✅ Contact messages management
- ✅ Testimonials management
